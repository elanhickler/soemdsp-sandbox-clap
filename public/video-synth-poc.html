Video Synth Raster — proof of concept

A Lorenz attractor's continuous state is walked one step per pixel (row-major raster order), producing a grayscale frame. Brightness is blended with the previous frame using the same one-pole decay model documented in the soemdsp-sandbox-phosphor fork. Runs on its own requestAnimationFrame clock, not the audio engine — this is not wired into the node graph.
